# [Simple Calendar]


## 특징
- #### 개인 일정, 업무 일정 등 일정을 분리하고 싶을 때 사용
- #### 공휴일만 간단히 보여주며 약 15자 내의 메모와 날짜별 배경색 조절을 통한 간단한 일정 기획


## 원리/기술
- #### HTML, CSS로 이미지 없이 자체 기능으로 디자인을 모두 해결
- #### 공공데이터포털 Open API를 통한 공휴일 정보 로드
- #### 캐시를 통해 기록한 메모를 저장
- #### TWA로 웹앱을 구현해 안드로이드 UX 최적화

## 향후 발전 계획
- #### 프로필 추가/변경 기능으로 더 여러 일정을 관리
- #### 캐시 방식이 아닌 로그인 회원 방식으로 교체해 기기 환경의 제약 없이 기존의 일정을 로드
- #### 일정 주기마다 돌아올 일정은 자동으로 일정 간격마다 추가할 수 있도록 일정 자동 추가
- #### 화이트 모드/다크 모드 테마 변경
- #### 팝업 알림 기능으로 일정이 다가올 때 알려주기
- #### 달력 로딩 로직 최적화로 날짜 이동 중 끊김 삭제
